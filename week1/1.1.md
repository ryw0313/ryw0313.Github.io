# 作业：C语言编程和优化

```
/*loop1*/
for(i=0;i<N;i++){
    a[i]=a[i]*2000;
}

/*loop2*/
b=a;
for(i=0;i<N;i++){
    *b=*b *2000;
}
哪种更好
```

* 测试代码，撰写文档，给出定量结果
* 补充代码，形成完整程序
* 思考完整测试方案，不同问题规模，不用编译器优化级别，不同编译器，不同机器
